# Dante SOCKS proxy

**SOCKS proxy with user/password authorization enabled.**

Based on: https://hub.docker.com/r/vimagick/dante/


## Available env vars

- `SOCKS_USER` - username of proxy user
- `SOCKS_PASSWORD` - password for user
